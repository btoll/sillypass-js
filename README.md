# Sillypass

## Installation

`npm install https://github.com/btoll/sillypass.git -g`

## Usage

    Command | Description
    ------------ | -------------
    generate | Generate a password.

    Option | Description
    ------------ | -------------
    -n=NUM | The length of the password (defaults to 12 characters).
    -h, --help | Display help.
